## Sobre o Projeto

Este repositório contém o desenvolvimento do **Estudo Dirigido** da disciplina de Inteligência Artificial.

O projeto tem como objetivo implementar e analisar agentes inteligentes, relacionando os conceitos teóricos estudados em sala de aula com uma aplicação prática.

Ao longo do desenvolvimento, este repositório será atualizado com:

- Implementação do ambiente;
- Implementação dos agentes inteligentes;
- Código-fonte;
- Documentação;
- Resultados obtidos;
- Relatório e materiais de apoio.
